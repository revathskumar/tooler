Tooler

Ruby Project boilerplater

What it does

  • Generate minimum code need to run
  • Add README.md
  • Add License
  • Add .gitignore
  • Add Gemfile
  • Add Procfile

Install

$ gem install tooler

Usage

$ cd project/directory
$ tooler [cuba/sinatra]
$ thin -R config.ru start

Current Support

  • Cuba
  • Sinatra
